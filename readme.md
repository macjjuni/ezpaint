## 📌 Intro

맥 OS "미리보기" 기본 앱 사용이 불편한 나머지 쉽게 사용하기 위해 직접 만들어 본 그림판 웹앱

<br>

## 📌 Function

- [x] 그림 그리기 & 자르기
- [x] 작업 되돌리기(Ctrl + z)
- [x] 단축키를 통한 이미지 복사, 붙여넣기
- [x] 이미지 다운로드

<br>

## Package Install

```
yarn
or
npm i
```

## Run dev

```
yarn dev
or
npm run start
```

## Run build

```
yarn build
or
npm run build
```
